## Flaskでフォームから送られてきた、コンテントタイプがapplication/x-www-form-urlencodedの場合の値の取得方法

**Flask**ではフォームデータを`request.form["key"]`から取得できます。
**Gin**では`c.PostForm("key")`

```python

@app.route("/user", methods=["POST"])   # ← POST メソッドを受け付ける
def get_request_param():
    user_id = request.form["id"]         # ← フォーム送信された "id"
    return "<html><body><h1>リクエストパラメータ:" + user_id + "</h1></body></html>"

```
