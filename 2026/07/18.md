## GoのHTTP標準パッケージ

Goの標準HTTPパッケージの`net/http`はリクエストが来るたびに自動で新しいgoroutineを起動してハンドラを実行している。

```go
func handler(w http.ResponseWriter, r *http.Request) {
    // この関数はリクエストごとに別のgoroutineで実行される
    users := fetchUsersFromDB() // ブロックしてOK
    json.NewEncoder(w).Encode(users)
}

func main() {
    http.HandleFunc("/api/users", handler)
    http.ListenAndServe(":8080", nil)
}
```

他のAPIを開発する例では`Hono`はシングルスレッドで動くので、DBに接続してデータを取得するような重い処理は`async/await`を使って明示的に実行権を手放す必要がありますが、Goはリクエストごとに別々のgoroutineが動きますので動機的なコードを書くことができます。
