## Python の標準入力の受け取り方
---
**1行に1つの数**:
```python
n = int(input())
```
**一行に複数の数**:
```python
a, b = map(int, input().split())
```
**1行に配列**
```python
nums = list(map(int, input().split()))
```
**N行に1つずつ**:
```python
n = int(Input())
nums = [int(input()) for _ in range(n)]
```
**文字列**:
```python
s = input()
```

Hono のルーティング問題
---
Honoは重複するルートを登録順でマッチさせるため、動的ルートの後ろに静的ルートを登録してしまうと、静的ルートにいくはずのリクエストが、すべて動的ルートに受け取られてしまう。
