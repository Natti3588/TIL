## 学んだこと

### Empty State
---
**エンプティステート（empty state）**とは UI において「表示するデータやコンテンツが何もない状態」のことを言います。

###### Empty State が表示される流れ

1. リクエスト: ユーザーが検索を実行し、フロントエンドがサーバーに検索リクエストを送る。
2. データベース検索: バックエンドがフロントエンドからリクエストを受け取り、データベースから値を取得する
3. レスポンス: データベースに該当するデータがなかった場合、エラーではなく「空のデータ」をエラーではなく正常なレスポンスとして、フロントエンドに返す
4. フロントエンドが「空のデータ」を受け取り、「表示するデータ」がないと判断して、「検索結果が0件です」を UI に表示させる。

-> **この「データがないときの画面表示（UIの状態）」を Empty State を呼ぶ**
